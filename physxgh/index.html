PhysX.GH

About

PhysX.GH is a rigid body simulation tool for Grasshopper. It uses open source real-time physics engine NVIDIA PhysX and a C# wrapper from stilldesign/PhysX.Net. It runs with GPU and provides users with fast simulation results.

Download: https://www.food4rhino.com/app/physxgh

Forum: https://www.grasshopper3d.com/group/physx-gh
